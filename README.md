# 小资金商品期货·模拟盘跟踪系统

一个纯静态的网页版商品期货模拟盘跟踪系统：观察池 → 基本面 → 三因子信号 → 模拟交易 → 资金曲线。

- 起始资金：¥15,000（可调）
- 目标：¥1,000,000
- 数据全部保存在浏览器 `localStorage`，请定期使用“导出数据”备份

## 在线访问

https://lee20260315.github.io/futures-tracker/

## 本地运行

直接用浏览器打开 `index.html`，或运行本地静态服务器：

```
python -m http.server 8000
```

然后访问 http://localhost:8000/

## 主要功能

1. **品种观察池** - 13 个商品期货品种，自动/手动维护现价、历史分位、成本线
2. **基本面跟踪** - 每个品种 5 维度评分（1-5）
3. **三因子信号引擎** - 估值底 + 供给收缩 + 催化剂 → 甜点级/轻仓/观望
4. **模拟交易** - 开仓/平仓、自动计算保证金、风险占比、浮动盈亏
5. **交易日志** - 复盘、纪律执行率统计
6. **资金曲线** - 累计收益、最大回撤、胜率、盈亏比、距 100 万进度

## 行情数据源

- 主源：东方财富 `push2.eastmoney.com`（HTTPS，单品种 JSONP）
- 备源：新浪期货 `hq.sinajs.cn`（nf_xxx0 连续合约）
- 失败时回退手动输入
